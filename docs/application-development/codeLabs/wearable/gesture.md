### Smart Gesture  
This app demonstrates and tests various gesture and crown smartGesture is a lightweight wearable demo app for Huawei Watch 5, designed to demonstrate and test touch based gesture, crown (button) interactions and smart gestures.
It helps developers verify the systemâ€™s response to different operations such as taps, swipes, button clicks, and long presses.

#### Software requirements
- **DevEco Studio version**: DevEco Studio 5.1.0 Release or higher version
- **HarmonyOS SDK version**: API version 18

- **Development watch**: HUAWEI Watch 5
- **HarmonyOS system**: 5.1.0 Release or higher version
